Blockchain Networks

Level 1 networks are the foundational blockchains that support the entire ecosystem of cryptocurrencies. They provide the infrastructure for transactions, smart contracts, and decentralized applications (dApps). These are massive investment instruments, because there are hundreds of applications built on top of each of such network. Typically, you should not worry whether they are limited supply or not, since all networks have sophisticated management procedures.

Utility Tokens for Crypto Exchanges

Many crypto exchanges have their own native tokens. Regardless of the reasons why they do it, there are billions of dollars in these tokens, and they are very reliable and stable. Exchanges host mobile apps and web sites and they have ability to create huge demand for their tokens, and all this drives the price up. Typically, you should not worry whether they are limited supply or not, since all exchanges worry about their prestige, they do buy back or burning, or something similar.

Token Name Description
BNB Binance Coin ($BNB) The native cryptocurrency of the Binance exchange, BNB is used to pay for trading fees and participate in token sales on the Binance platform.
CRO Cronos ($CRO) The native token of the Crypto.com ecosystem, used for staking, rewards, and transaction fees within the platform.
OKB OKB ($OKB) The native token of the OKX exchange, used for trading fee discounts, staking, and ecosystem rewards.

DePIN Tokens

DePIN (Decentralized Physical Infrastructure Networks) tokens are used to incentivize participation in decentralized networks. This is a very specific type of tokens, they are used to distribute rewards to users instead of distributing dividends to shareholders, or cash rewards. Usually, such tokens are limited supply, and there are buybacks. Think of it as a form of paying dividends.

Token Name Description
DIMO DIMO ($DIMO) A token that rewards users for sharing car data through connected devices.
HNT Helium ($HNT) The native token of the Helium network, used to reward users for providing wireless coverage.
MMT MapMetrics ($MMT) A token that rewards users for sharing location data to build a decentralized navigation system.
ATH Aethir ($ATH) A token for decentralized cloud computing, rewarding users for providing GPU-optimized compute power.
GEO Geodnet ($GEO) A token for the decentralized geospatial network, rewarding users for deploying Geodnet miners.
SLC Silencio ($SLC) A token that rewards users for measuring noise levels and contributing to decentralized noise data collection.
WIFI WiFi ($WIFI) A token that rewards users for sharing their WiFi networks to create a decentralized internet infrastructure.
ROAM Roam ($ROAM) A token that incentivizes users for contributing to decentralized location-based services and roaming networks.
AIOZ AIOZ ($AIOZ) A token for decentralized content delivery networks, rewarding users for sharing bandwidth and storage.
AUTONOMI Autonomi ($AUTONOMI) A token that rewards users for contributing to decentralized IoT networks and automation systems.
FIL Filecoin ($FIL) A decentralized storage network token that rewards users for providing storage space.
GLM Golem ($GLM) A token for decentralized compute networks, rewarding users for sharing their computing power.
LPT Livepeer ($LPT) A token for decentralized video streaming networks, rewarding users for transcoding and distributing video.
MYST Mysterium ($MYST) A token for decentralized VPN services, rewarding users for sharing their internet bandwidth.
DVPN Sentinel ($DVPN) A token for decentralized VPN networks, rewarding users for providing secure and private internet access.

Utility Meme Tokens

Meme tokens have evolved beyond their humorous origins to provide real-world utility. These tokens often combine community-driven initiatives with unique features that set them apart from traditional cryptocurrencies.

Token Name Description
DOGE Dogecoin ($DOGE) Initially created as a joke, Dogecoin has become a widely accepted cryptocurrency for tipping, donations, and microtransactions due to its low fees and fast transaction speeds.
FLOKI Floki Inu ($FLOKI) Floki Inu combines meme culture with utility by offering features like FlokiFi (a DeFi ecosystem), an NFT marketplace, and educational platforms for crypto enthusiasts.
SHIB Shiba Inu ($SHIB) Shiba Inu has grown from a meme token to a utility token with features like ShibaSwap (a decentralized exchange), NFT integration, and staking rewards.

Lending Platforms

Lending platforms enable users to lend or borrow cryptocurrencies, often using decentralized protocols. These platforms utilize tokens for governance, staking, rewards, and other incentives.

Token Name Description
DAI Dai ($DAI) A decentralized stablecoin pegged to the US dollar, used for lending, borrowing, and trading on DeFi platforms.
AAVE Aave ($AAVE) A governance token for the Aave protocol, used for staking, governance, and earning rewards.
COMP Compound ($COMP) A governance token for the Compound protocol, used to incentivize users and participate in protocol decisions.
MKR MakerDAO ($MKR) A governance token for MakerDAO, used to manage the stability mechanisms of the DAI stablecoin.
RAY Raydium ($RAY) A token for the Raydium protocol on Solana, used for governance and providing liquidity incentives.
